On Fractal Programming

Fractal Programming is only possible with tools that provide a sufficiently robust framework for expressing the rules used to expand the fractal. The resulting implementation is mathematically pure and predictable.

Fractal Code is very maintainable. Because all code follows the same fractal patterning, any errors that affect one such piece of code affect all such pieces of code, increasing the footprint for discovering the error in the interface or implementation of the code. Furthermore, corrections to such discovered errors become a permanent part of the rule base, providing the ultimate in ISO9000 process correction.
